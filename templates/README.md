# Templates Directory

## Overview
This directory contains boilerplate templates for common DayZ modding tasks.

## Available Templates

### Mod Structure Templates
Basic mod directory structures and configuration templates.
- See: [mod-structure/](mod-structure/)

### Killfeed Templates
Ready-to-use killfeed system templates.
- See: [killfeed-templates/](killfeed-templates/)

### Config Templates  
Configuration file templates (config.cpp, types.xml, etc.)
- See: [config-templates/](config-templates/)

### Script Templates
Common EnScript patterns and boilerplate code.
- See: [script-templates/](script-templates/)

## Usage
1. Browse the appropriate subdirectory
2. Copy the template you need
3. Customize for your specific use case
4. Follow the comments and instructions in each template
