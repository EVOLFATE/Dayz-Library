# Chernarus+ Interactive Map Guide

## Overview

**Chernarus+** is the flagship map of DayZ, based on the original Chernarus from Arma 2. It's a 225 km² post-Soviet state featuring diverse terrain, from coastal towns to dense forests and military installations.

## Map Specifications

### Technical Details
- **Size**: 225 km² (15,360m x 15,360m)
- **Coordinate System**: 0,0 (Southwest) to 15360,15360 (Northeast)
- **Grid System**: Letter-Number format (A1-O15)
- **Terrain Types**: 12 distinct biomes
- **Buildings**: 3,000+ enterable structures
- **Towns/Cities**: 50+ settlements
- **Military Bases**: 15+ locations
- **Elevation**: Sea level to 620m (Altar mountain)

## Map Coordinates Reference

### Grid System
```
Letters (West to East): A-O
Numbers (South to North): 1-15
Example: "M7" = Grid square M7
```

### Converting Coordinates
```
In-Game Coordinate → Grid:
X: 4520, Z: 10430 = Grid E11
Formula: Letter = X/1024, Number = Z/1024
```

### GPS Coordinates
DayZ uses format: `XXX YYY` (3-digit bearing)
Example: `042 104` = Northeast
